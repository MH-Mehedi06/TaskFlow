# TaskFlow

A full-stack TaskFlow-inspired home services marketplace.

## Stack
- **Frontend**: React 18 + Vite + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express + TypeScript
- **Database**: MongoDB + Redis
- **Payments**: Stripe
- **Real-time**: Socket.io
- **AI**: OpenAI GPT-4o

## Setup
1. `cd server && npm install && cp .env.example .env`
2. `cd client && npm install`
3. Fill in `.env` with your API keys
4. `cd server && npm run dev`
5. `cd client && npm run dev`
